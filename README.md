textlint-rule-imas-idol
=======================

[THE IDOLM@STER (アイドルマスター)](https://idolmaster-official.jp/)の登場人物(アイドル)の誤表記を
[textlint](https://textlint.github.io/)で検出するためのルールセットです。

[アイマスDB](https://imas-db.jp/)が管理を行っています。
